EvisToEe
~~~~~~~~

Description
^^^^^^^^^^^
Transform visible energy into the positron energy :math:`E_{\text{vis}} \rightarrow E_{\text{e}}`  value.

Inputs
^^^^^^

  1. Array of :math:`E_\text{vis}`.

Outputs
^^^^^^^

1. Array of :math:`E_{\text{e}}`.

Implementation
^^^^^^^^^^^^^^
.. math::
  E_{\text{e}} = E_{\text{vis}} - m_e
